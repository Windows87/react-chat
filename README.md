## To use

First clone the repository:
```bash
# Clone this repository
git clone https://github.com/Windows87/react-chat
# Go into repository
cd react-chat
```

Start the react app:
```bash
# Go into react app
cd front-end
# Install dependencies
npm install
# Start
npm start
```

And start the server:
```bash
# Go into server
cd back-end
# Install dependencies
npm install
# Start
npm start
```
